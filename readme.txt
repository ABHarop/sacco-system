HOW TO INSTALL
1. Copy the folder named 'sacco' and paste it in the htdocs folder in xampp or wamp folder in wamp.

2. Open the web browser and enter 'localhost' in the address bar.

3. Click phpMyAdmin and create a database with the name saccodb.

4. Import the database named saccodb.sql inside the database folder in the sacco folder.

5. Type localhost/sacco in the address bar.
